AKRA Documentation v0.3.0

To use documentation, start local server and navigate to index2.html;
To use live search start search server as follows: "node serach/search.js akra.json"

All logics is in "js/app.js". The file is separated into sections: app, controllers, logics, directives, filters;
Auxillary js bindings (left pane mousescroll, livesearch category hiders) are in file "js/document.js";

HTML templates are situated in "templates/";

Documentation data is situated in "data/";
